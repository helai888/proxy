# Proxy UI 版本快照

为了避免在同一个 `proxy.js` 上反复合并冲突，这里把不同阶段的版本拆成独立文件（后续继续新增版本文件，不在旧文件上直接改）：

- `proxy.v1-original.js`：初始原版（无新版 Dashboard 风格）
- `proxy.v2-themed.js`：仅加入新版主题风格与布局
- `proxy.v3-sidebar-nav.js`：加入侧边栏导航、页面切换与折叠逻辑
- `proxy.v4-sidebar-nav-modal.js`：在 v3 基础上，进一步调整导航样式、按钮统一渐变、节点编辑改为弹窗

使用方式：
1. 每次新需求都新增 `proxy.vX-*.js` 文件，不直接改旧版本。
2. 按需选择一个版本文件，将其内容覆盖到线上使用的 `proxy.js`。
3. 其他版本文件保持不动，避免再次在同一文件上产生冲突。
